Headers Configuration

Content-Security-Policy:
- frame-ancestors: 'self', *.github.com, github.com, *.vercel.app, vercel.app, about:blank
- script-src: Allows 'unsafe-inline' and 'unsafe-eval' for document.write

X-Frame-Options: ALLOWALL

Cross-Origin Policies:
- COEP: unsafe-none
- COOP: unsafe-none
- CORP: cross-origin

CORS: Enabled for all origins (*)
